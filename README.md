
```ShellScript
## create an npm package
##  go to the directory that should become a npm package
# npm init

## local install (only for this package)
# npm i http-server

## global install (operating system wide)
# sudo npm i -g http-server

## check where an app has been installed
# which http-server

## remove a package
# sudo npm r -g http-server

## npx - execute a package (binary)
# npx name-of-the-package
# npx http-server &! # (&! === start it in the background)
```
